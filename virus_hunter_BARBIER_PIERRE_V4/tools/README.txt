Process Explorer est un logiciel de Microsoft.
[] Ilpermet de surveiller les processus en cours d'exécution sur un système d'exploitation.
[] Il permet de voir les processus en cours d'exécution, leur utilisation des ressources système,
les fichiers et les dll qu'ils utilisent, les handles (objets) qu'ils ont ouverts,
les processus enfants et les services qui leur sont associés. 
[] Il permet également de tuer des processus et de changer la priorité des processus en cours d'exécution.
[] Il peut être téléchargé gratuitement à partir du site Web de Microsoft : 
https://docs.microsoft.com/en-us/sysinternals/downloads/process-explorer
______________________________________________________________________________________________________________________________________
Process Monitor est un autre outil de Microsoft
c'est un outil de surveillance de système développé par Microsoft.
[] Il permet de surveiller en temps réel toutes les activités de système, 
tels que les accès de fichiers, les registres, les processus et les réseaux. 
[] Il permet de voir les opérations en cours et les opérations récentes sur le système d'exploitation. 
[] Il permet également de filtrer les informations affichées pour afficher uniquement les opérations qui vous intéressent. 
[] Il peut être téléchargé gratuitement à partir du site Web de Microsoft :
https://docs.microsoft.com/en-us/sysinternals/downloads/procmon

VirusTotal est un service gratuit en ligne qui permet d'analyser des fichiers
et des URL à la recherche de virus, de logiciels malveillants et d'autres
menaces potentielles.
[] Il permet l'analyse de fichiers (jusqu'à 256 Mo) en utilisant plus de
70 moteurs antivirus.
[] Il permet l'analyse d'URL.
[] Il permet la recherche de fichiers malveillants.
[] Il permet de rendre un résultats d'analyse détaillés.
[] Il permet d'établir un rapports de confiance, car les utilisateurs peuvent 
également soumettre des fichiers ou des URL pour obtenir un rapport de confiance
qui indique si le fichier ou l'URL est sûr ou non.
[] Il permet la surveillance de la sécuritécar les utilisateurs peuvent surveiller
la sécurité de leur réseau en utilisant l'API VirusTotal pour analyser les fichiers
et les URL qui sont téléchargés ou accédés par leur réseau.
[] Il est consultable par l'url suivante : https://www.virustotal.com

Sigcheck est un outil en ligne de commande développé par Microsoft 
[] Il permet de vérifier les signatures numériques des fichiers, notamment les exécutables Windows,
les bibliothèques de liens dynamiques et les pilotes. 
[] Il peut être utilisé pour vérifier si un fichier est signé numériquement par un éditeur de confiance et si sa signature est valide.
[] le lien suivant permet d'accéder au instruction de téléchargement : https://docs.microsoft.com/en-us/sysinternals/downloads/sigcheck
 ________________________________________________________________________________________________________________________________________
Signtool est un outil en ligne de commande utilisé pour signer numériquement des fichiers sur les systèmes d'exploitation Linux et Unix.
[] Il utilise le standard de signature numérique de logiciels PKCS#7 pour créer et vérifier des signatures numériques. 
[] Le lien suivant permet d'accerder à la page de téléchargement des SDK : https://developer.microsoft.com/fr-fr/windows/downloads/windows-sdk/
